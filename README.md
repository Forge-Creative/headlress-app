# Headless project

## Tech stack
**Back end** - [WordPress](https://en-nz.wordpress.org/)

- [WPGraphQL](https://wordpress.org/plugins/wp-graphql/)

**Front end** - [Next.js](https://nextjs.org/)

- [Apollo client](https://www.apollographql.com/docs/react/)


## Installation

```sh
npm i
npm run dev
```
